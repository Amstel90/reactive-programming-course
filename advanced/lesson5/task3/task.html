Предоставте ручную имплементацию контроля потока которая будет соответсвовать следующей логике:
  • Первый запрос должен соответсвовать 1
  • Каждый последующий запрос должен быть удвоен
  • Следующий запрос должен быть отправлен только в случае получения уже запрошенных элементов
Используйте BaseSubscriber для имплементации собтвенного подписчика
